Experience

Tower Conquest is a project developed and released by another studio which changed ownership post launch. I worked on the project by supporting the game with live ops. As a finished product, I mostly worked within the design constraints of the game. My responsibilities involved designing new daily offers, analyzing their performances and updating them accordingly. My other responsibilities was to design a new game mode for live ops, the result of which is the Daily Challenge feature.

 

Along with updating the current game, I directed the design for its sequel with major changes to the core progression system and the meta-game. The goal was to improve meta-game of the existing game, increase the depth of the game for long term player engagement and strengthening the KPIs further.

 

This is my first mobile game in RPG genre and a great hands-on experience of live ops. Expanding the game with the sequel and designing core RPG features has been the key challenge I was able to take on this project. Its has help me understand how deep progression systems can be designed which are core to the game and how meta game can be build around this progression system.
